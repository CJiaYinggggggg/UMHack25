Overview

We combine regime detection (HMM) with sequential forecasting (LSTM) to generate entry and exit signals:

  1. Regime Filter: Use HMM to identify Bull/Bear periods.
  2. Signal Generator: Train LSTM on rolling windows of returns during Bull states.
  3. Execution: Only take long trades if HMM indicates Bull; otherwise stay flat or hedge.
  4. Risk Management: Fixed stop-loss at 2% and take-profit at 5% per trade.

Walkthrough

1. Load price data and fit HMM on returns.
2. Label each time-step with a regime.
3. Train LSTM only on data labeled Bull.
4. During live trading, predict next-step price with LSTM; if predicted return > threshold, go long.
