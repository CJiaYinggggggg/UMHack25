What is Backtesting?

Backtesting simulates a trading strategy on historical data to measure its performance before deploying it live. It helps identify strengths, weaknesses, and parameter sensitivities.

Core Components

  • Data Loader: Fetches and preprocesses time-series data.
  • Strategy Engine: Generates buy/sell signals based on rules or models.
  • Execution Simulator: Applies signals to compute P&L with realistic assumptions (slippage, fees).
  • Metrics Calculator: Computes return, Sharpe ratio, drawdowns, etc.

Pseudocode Example

// Hardcoded backtest function
function runBacktest(data, strategy) {
  const results = [];
  data.forEach((point, idx) => {
    const signal = strategy.generateSignal(data.slice(0, idx + 1));
    const pnl = point.price * signal; // simplistic
    results.push({ date: point.date, signal, pnl });
  });
  return results;
}
